Varincia da Soma

A hiptese nula afirma que a diferena de probabilidade entre os dois grupos zero. Portanto, a mdia para essa distribuio normal ser zero. A outra propriedade de que precisaremos para a distribuio normal o desvio padro ou a varincia.

Observao: a varincia o desvio padro ao quadrado. A varincia da diferena depender das varincias da probabilidade para ambos os grupos.

Verificando a Hipótese Nula e a Hipótese Alternativa¶

Definio da hiptese nula e da hiptese alternativa:

A hiptese nula a posio de que a mudana no design feito para o grupo de teste resultaria em nenhuma mudana na taxa de converso.

A hiptese alternativa a posio oposta de que a mudana no design do grupo de teste resultaria em uma melhoria (ou reduo) na taxa de converso.

A hiptese nula ser uma distribuio normal com uma mdia de zero e um desvio padro igual ao erro padro agrupado.

A hiptese alternativa tem o mesmo desvio padro que a hiptese nula, mas a mdia estar localizada na diferena na taxa de converso, d_hat. Isso faz sentido porque podemos calcular a diferena nas taxas de converso diretamente dos dados, mas a distribuio normal representa possveis valores que nosso experimento poderia ter nos dado.

Frmula para o clculo de z:

$$ z = \frac{(\bar{x_1}-\bar{x_2})-D_0}{\sqrt{\sigma_1^{2}/n_1+\sigma_{2}^{2}/n_2}} $$$$ z = \frac{(\hat{p_1}-\hat{p_2})-0}{\sqrt{\hat{p}\hat{q}(\frac{1}{n_1}+\frac{1}{n_2})}}$$

# Funo que retorna o erro padro agrupado para 2 amostras
def erro_padrao_agrupado_func(N_A, N_B, X_A, X_B):
    p_hat = prob_agrupada_func(N_A, N_B, X_A, X_B)
    SE = np.sqrt(p_hat * (1 - p_hat) * (1 / N_A + 1 / N_B))
    return SE

# Retorna o valor z para um determinado nvel de significncia
def z_val(sig_level = 0.05, two_tailed = True):
    
    # Gera distribuio para o valor z
    z_dist = scs.norm()
    
    # Verifica se devemos checar as duas caudas
    if two_tailed:
        sig_level = sig_level/2
        area = 1 - sig_level
    else:
        area = 1 - sig_level

    # Valor de z
    z = z_dist.ppf(area)

    return z
In [146]:
# Calcula o intervalo de confiana
def confidence_interval(sample_mean = 0, sample_std = 1, sample_size = 1, sig_level = 0.05):
    
    # Calcula o valor de z
    z = z_val(sig_level)

    # Limites  esquerda e direita
    left = sample_mean - z * sample_std / np.sqrt(sample_size)
    right = sample_mean + z * sample_std / np.sqrt(sample_size)

    return (left, right)

# Funo para o plot de uma distribuio normal
def plot_norm_dist(ax, 
                   mu, 
                   std, 
                   with_CI = False, 
                   sig_level = 0.05, 
                   label = None):

    # Gera valores para a varivel aleatria x
    x = np.linspace(mu - 12 * std, mu + 12 * std, 1000)
    
    # Cria a distribuio normal
    y = scs.norm(mu, std).pdf(x)
    
    # Plot
    ax.plot(x, y, label = label)

    # Se tivermos intervalo de confiana, inclumos no plot
    if with_CI:
        plot_CI(ax, mu, std, sig_level = sig_level)

# Funo que preenche entre o limite de significncia superior e a distribuio para hiptese alternativa
def show_area(ax, d_hat, stderr, sig_level):

    # Intervalo de confiana
    left, right = confidence_interval(sample_mean = 0, sample_std = stderr, sig_level = sig_level)
    
    # Valores para x
    x = np.linspace(-12 * stderr, 12 * stderr, 1000)
    
    # H0
    null = ab_dist(stderr, 'controle')
    
    # H1
    alternative = ab_dist(stderr, d_hat, 'teste')

    # Se o tipo da rea for igual a power
    # Preenchemos entre o limite de significncia superior e a distribuio para hiptese alternativa
    ax.fill_between(x, 0, alternative.pdf(x), color = 'green', alpha = 0.25, where = (x > right))
    ax.text(-3 * stderr, null.pdf(0), 'power = {0:.3f}'.format(1 - alternative.cdf(right)), 
                fontsize = 12, ha = 'right', color = 'k')

# Funo que retorna um objeto de distribuio dependendo do tipo de grupo
def ab_dist(stderr, d_hat = 0, group_type = 'controle'):

    # Verifica o tipo de grupo
    if group_type == 'controle':
        sample_mean = 0
    elif group_type == 'teste':
        sample_mean = d_hat

    # Cria uma distribuio normal que depende da mdia e do desvio padro
    dist = scs.norm(sample_mean, stderr)
    return dist

A rea sob a curva da hiptese alternativa igual a 1. Se o design alternativo (sem avaliaes) for realmente melhor, o poder a probabilidade de aceitarmos a hiptese alternativa e rejeitarmos a hiptese nula e igual rea sombreada em verde (verdadeiro positivo). A rea oposta sob a curva alternativa a probabilidade de no rejeitarmos a hiptese nula e rejeitarmos a hiptese alternativa (falso negativo). Isso conhecido como beta no teste A/B ou teste de hiptese e mostrado abaixo.

Se a hiptese nula for verdadeira e realmente no houver diferena entre os grupos de controle e teste, o nvel de significncia a probabilidade de rejeitarmos a hiptese nula e aceitarmos a hiptese alternativa (falso positivo). Um falso positivo quando conclumos erroneamente que o novo design melhor. Este valor baixo porque queremos limitar essa probabilidade.

Muitas vezes, um problema ser fornecido com um nvel de confiana desejado em vez do nvel de significncia. Um nvel de confiana tpico de 95% para um teste A / B corresponde a um nvel de significncia de 0,05.

# Funo para incluir o valor z no plot
def zplot(area = 0.95, two_tailed = True, align_right = False):

    # Cria a rea de plotagem
    fig = plt.figure(figsize = (12, 6))
    ax = fig.subplots()
    
    # Cria a distribuio normal
    norm = scs.norm()
    
    # Cria os pontos de dados para o plot
    x = np.linspace(-5, 5, 1000)
    y = norm.pdf(x)

    ax.plot(x, y)

    # Cdigo para preencher reas para testes bidirecionais
    if two_tailed:
        left = norm.ppf(0.5 - area / 2)
        right = norm.ppf(0.5 + area / 2)
        ax.vlines(right, 0, norm.pdf(right), color = 'grey', linestyle = '--')
        ax.vlines(left, 0, norm.pdf(left), color = 'grey', linestyle = '--')

        ax.fill_between(x, 0, y, color = 'grey', alpha = 0.25, where = (x > left) & (x < right))
        
        plt.xlabel('z')
        plt.ylabel('PDF')
        plt.text(left, norm.pdf(left), "z = {0:.3f}".format(left), 
                 fontsize = 12, 
                 rotation = 90, 
                 va = "bottom", 
                 ha = "right")
        plt.text(right, norm.pdf(right), "z = {0:.3f}".format(right), 
                 fontsize = 12, 
                 rotation = 90, 
                 va = "bottom", 
                 ha = "left")
    
    # Para testes de uma cauda
    else:
        # Alinhamento  direita
        if align_right:
            left = norm.ppf(1-area)
            ax.vlines(left, 0, norm.pdf(left), color = 'grey', linestyle = '--')
            ax.fill_between(x, 0, y, color = 'grey', alpha = 0.25, where = x > left)
            plt.text(left, norm.pdf(left), "z = {0:.3f}".format(left), 
                     fontsize = 12, 
                     rotation = 90, 
                     va = "bottom", 
                     ha = "right")
        
        # Alinhamento  esquerda
        else:
            right = norm.ppf(area)
            ax.vlines(right, 0, norm.pdf(right), color = 'grey', linestyle = '--')
            ax.fill_between(x, 0, y, color = 'grey', alpha = 0.25, where = x < right)
            plt.text(right, norm.pdf(right), "z = {0:.3f}".format(right), 
                     fontsize = 12, 
                     rotation = 90, 
                     va = "bottom", 
                     ha = "left")

    # Inclumos texto no plot
    plt.text(0, 0.1, "rea Sombreada = {0:.3f}".format(area), fontsize = 12, ha = 'center')
    
    # Labels
    plt.xlabel('z')
    plt.ylabel('PDF')

    plt.show()

# Funo para encontrar o tamanho mnimo da amostra
def calcula_tamanho_min_amostra(N_A, 
                                N_B, 
                                p_A, 
                                p_B, 
                                power = 0.8, 
                                sig_level = 0.05, 
                                two_sided = False):
   
    k = N_A/N_B
    
    # Distribuio normal para determinar os valores z
    standard_norm = scs.norm(0, 1)

    # Encontramos o valor de z para o poder estatstico
    Z_beta = standard_norm.ppf(power)
    
    # Encontramos z alfa
    if two_sided == True:
        Z_alpha = standard_norm.ppf(1-sig_level/2)
    else:
        Z_alpha = standard_norm.ppf(1-sig_level)

    # Probabilidade agrupada
    pooled_prob = (p_A + p_B) / 2

    # Tamanho mnimo da amostra
    min_N = (2 * pooled_prob * (1 - pooled_prob) * (Z_beta + Z_alpha)**2 / efeito_minimo**2)    

    return min_N
